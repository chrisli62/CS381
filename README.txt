How to run language_modeling.py
*******************************

[1] Create a directory and place all files from language_modeling.zip into the same directory
[2] On an IDE, run the main file called "language_modeling.py". If using command prompt or a terminal enter "py language_modeling.py"
    or "python language_modeling.py" respectively.
[3] Results will appear in the IDE console or command propt or terminal (!!!NOTE: Program will take time to return results!!!)
